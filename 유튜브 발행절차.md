# 유튜브 발행 절차

> 기준일: 2026-06-15  
> 현재 상태: **Mock 모드** (실 OAuth 미연결)

---

## 전체 발행 흐름

```
MP3 업로드 → 플레이리스트 구성 → 영상 렌더링 → 썸네일 생성 → YouTube 발행
```

---

## 단계별 절차

### 1단계 — MP3 업로드

- 화면 중앙 드래그&드롭 영역 또는 파일 선택 버튼으로 음원 파일 추가
- 지원 형식: `.mp3` `.wav` `.flac` `.aac` `.m4a`
- API: `POST /tracks/upload` (multipart/form-data)
- 업로드된 파일은 `uploads/` 디렉토리에 UUID 파일명으로 저장됨

### 2단계 — 플레이리스트 구성

- 왼쪽 사이드바에서 플레이리스트 선택 또는 + 버튼으로 생성
- 트랙 편집 테이블에서 순서(sort_order), 제목, 아티스트, 앨범 인라인 수정 가능
- API: `POST /playlists`, `PUT /tracks/{id}`

### 3단계 — 영상 렌더링

- 트랙 편집 섹션의 **렌더링** 버튼 클릭
- FFmpeg로 음원을 연결하고 단색 배경(기본값 `#12372a`)과 합성하여 MP4 생성
- 챕터 타임스탬프 자동 계산 후 설명란에 삽입
- API: `POST /video/render`
- 결과 파일 저장 위치: `renders/playlist-{id}-{uuid}.mp4`
- **⚠️ FFmpeg 미설치 시 RuntimeError 발생** → `winget install Gyan.FFmpeg` 후 재시도

### 4단계 — 썸네일 생성 (선택)

- 출력 패널의 **썸네일** 버튼 클릭
- Pillow로 1280×720 이미지 생성 (스타일 고정: `studio`)
- API: `POST /thumbnail/generate`
- 결과 파일 저장 위치: `thumbnails/thumbnail-{video_id}-{uuid}.jpg`

### 5단계 — YouTube 발행

우측 패널에서 발행 정보 입력 후 **발행** 버튼 클릭:

| 필드 | 설명 |
|------|------|
| 영상 제목 | YouTube 영상 제목 |
| 설명 | 본문 (챕터 타임스탬프 자동 포함) |
| 태그 | 쉼표 구분 키워드 |
| 공개범위 | `private` / `unlisted` / `public` |
| 예약 시각 | 입력 시 예약 발행, 비워두면 즉시 발행 |

- API: `POST /youtube/upload`

## 현재 Mock 모드 동작

`.env`에 `YOUTUBE_UPLOAD_MODE=mock` 설정 시:

- 실제 YouTube에 업로드하지 않음
- `mock-uploaded-xxxxxxxxxxxx` 또는 `mock-scheduled-xxxxxxxxxxxx` 형태의 가짜 ID 반환
- DB의 Video 레코드 status가 `uploaded` 또는 `scheduled`로 갱신됨

## 실 YouTube OAuth 연동 절차

### 사전 준비

1. [Google Cloud Console](https://console.cloud.google.com) 에서 프로젝트 생성
2. **YouTube Data API v3** 활성화
3. OAuth 2.0 클라이언트 ID 생성 (유형: 데스크톱 앱 또는 웹 앱)
4. Refresh Token 발급

### Refresh Token 발급 방법

```bash
# google-auth-oauthlib 설치
pip install google-auth-oauthlib

# OAuth 인증 후 refresh_token 출력
python - <<'EOF'
from google_auth_oauthlib.flow import InstalledAppFlow

SCOPES = ["https://www.googleapis.com/auth/youtube.upload"]
flow = InstalledAppFlow.from_client_secrets_file("client_secrets.json", SCOPES)
creds = flow.run_local_server(port=0)
print("refresh_token:", creds.refresh_token)
EOF
```

### 환경변수 설정 (`.env`)

```env
YOUTUBE_UPLOAD_MODE=real
YOUTUBE_CLIENT_ID=your-client-id.apps.googleusercontent.com
YOUTUBE_CLIENT_SECRET=your-client-secret
YOUTUBE_REFRESH_TOKEN=your-refresh-token
```

### 백엔드 어댑터 구현 필요

`backend/app/services/youtube.py`의 `NotImplementedError` 부분에 실제 업로드 코드 작성:

```python
from googleapiclient.discovery import build
from googleapiclient.http import MediaFileUpload
from google.oauth2.credentials import Credentials

def _build_youtube_client(settings):
    creds = Credentials(
        token=None,
        refresh_token=settings.youtube_refresh_token,
        client_id=settings.youtube_client_id,
        client_secret=settings.youtube_client_secret,
        token_uri="https://oauth2.googleapis.com/token",
    )
    return build("youtube", "v3", credentials=creds)
```

## API 요청 예시 (발행 단계)

```bash
# YouTube 업로드 (즉시 발행)
curl -X POST http://127.0.0.1:8000/youtube/upload \
  -H "Content-Type: application/json" \
  -d '{
    "video_id": 1,
    "title": "Late Night Focus Mix",
    "description": "자동 생성된 플레이리스트입니다.\n\n챕터:\n00:00 Track 1\n03:45 Track 2",
    "tags": ["music", "playlist", "focus"],
    "privacy_status": "private",
    "publish_at": null
  }'

# YouTube 예약 발행
curl -X POST http://127.0.0.1:8000/youtube/upload \
  -H "Content-Type: application/json" \
  -d '{
    "video_id": 1,
    "title": "Late Night Focus Mix",
    "description": "...",
    "tags": ["music"],
    "privacy_status": "public",
    "publish_at": "2026-06-20T18:00:00+09:00"
  }'
```

## 발행 후 Video 상태 전이

```
draft → rendering → ready → uploaded
                          ↘ scheduled  (예약 발행 시)
                    ↘ failed     (렌더링 실패 시)
```

## 미구현 항목 (추후 개발 필요)

| 항목 | 비고 |
|------|------|
| YouTube OAuth 실 연동 | `services/youtube.py` 어댑터 작성 필요 |
