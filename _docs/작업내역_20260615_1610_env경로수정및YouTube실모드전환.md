## 작업 내용

YouTube 업로드가 mock 모드로 동작하는 원인을 파악하고 수정하였다.

## 원인

`config.py`의 `env_file=".env"` 가 상대경로여서, 백엔드를 `backend/` 디렉토리에서 실행하면
프로젝트 루트의 `.env`를 찾지 못하고 기본값(mock)을 사용하고 있었다.

## 변경된 파일

- `backend/app/config.py`
  - `env_file=".env"` → `env_file=str(PROJECT_ROOT / ".env")` (절대경로)
  - `extra="ignore"` 추가 (`.env`의 AWS 등 미정의 필드 무시)

## 결과

- `YOUTUBE_UPLOAD_MODE=real` 정상 로드
- `YOUTUBE_CLIENT_ID`, `YOUTUBE_REFRESH_TOKEN` 정상 로드
- 백엔드 재시작 완료
