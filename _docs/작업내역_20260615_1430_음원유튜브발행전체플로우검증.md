## 발견된 이슈

### DB 분리 문제 (⚠️ 주의)
- 포트 8000 (현재 메인 백엔드): `backend/data/app.db` 사용, 플레이리스트 0개
- 포트 8001 (테스트 백엔드): temp 경로 DB 사용, 실제 데이터 보유
- `.env`의 `DATABASE_URL=sqlite:///./data/app.db`가 상대경로라 실행 디렉토리에 따라 달라짐
- **임시 조치**: 포트 8001로 검증. 실운영 시 DB 경로를 절대경로로 변경 권장

### YouTube 실제 모드 주의
- `.env`에 `YOUTUBE_UPLOAD_MODE=real` + 실제 인증 정보가 있음
- 포트 8000 백엔드에서 `/youtube/upload` 호출 시 실제 YouTube 발행됨
- 테스트 시 mock 모드(`YOUTUBE_UPLOAD_MODE=mock`)로 변경 후 진행 권장

### 프론트엔드 React Client Manifest 오류
- 파일 수정 후 Next.js dev 서버가 오래 떠 있으면 manifest 오류 발생
- 해결: `npm run dev` 재시작으로 정상화됨
