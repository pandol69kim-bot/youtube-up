## 3. 음원 업로드 화면 작동 체크

```powershell
# 포트 정리
Get-NetTCPConnection -LocalPort 8000,3000 -State Listen | Stop-Process

# 백엔드 시작
python -m uvicorn app.main:app --host 127.0.0.1 --port 8000

# 프론트엔드 시작
cmd /c "cd frontend && npm run dev"

# 테스트용 WAV 파일 생성 (1초 사인파)
python -c "
import wave, struct, math
with wave.open('d:/tmp_test.wav', 'w') as f:
    f.setnchannels(1); f.setsampwidth(2); f.setframerate(44100)
    samples = [int(32767 * math.sin(2 * math.pi * 440 * i / 44100)) for i in range(44100)]
    f.writeframes(struct.pack('<' + 'h' * len(samples), *samples))
"

# 플레이리스트 생성
POST http://127.0.0.1:8000/playlists
Body: {"title":"테스트 플레이리스트","description":"업로드 테스트용"}
# 결과: {"id":1, "title":"테스트 플레이리스트"}

# WAV 파일 업로드 (Python urllib multipart)
POST http://127.0.0.1:8000/tracks/upload
Form: playlist_id=1, files=test-track.wav
# 결과: HTTP 200 [{"id":1,"title":"test-track","duration":0.0,...}]

# 비정상 케이스 검증
# - 비지원 포맷(bad.exe) → HTTP 400
# - 존재하지 않는 playlist_id=9999 → HTTP 404
```

## 4. YouTube OAuth2 refresh_token 발급

```powershell
# 필요 패키지 확인
python -m pip show google-auth-oauthlib google-api-python-client

# 실제 redirect_uri 확인 스크립트
python check_redirect_uri.py
# 결과: redirect_uri 실제 요청값: http://127.0.0.1:8080/

# OAuth 인증 스크립트 실행 (브라우저 열림)
python get_refresh_token.py
# 브라우저에서 Google 계정 로그인 → YouTube 권한 허용
# 결과: refresh_token 발급 및 .env 자동 저장
```

**발급된 refresh_token** → `.env`의 `YOUTUBE_REFRESH_TOKEN` 에 저장

**문제 해결 과정**:
- 1차 오류: `redirect_uri_mismatch` → Google Cloud Console에 `http://localhost:8080/` 추가
- 2차 오류: 여전히 mismatch → `localhost` 대신 `http://127.0.0.1:8080/` 로 변경
- 3차 오류: `UnicodeEncodeError` (이모지 ✅ → cp949 인코딩 불가) → 이모지 제거 후 재실행
- 최종 성공

## 6. FFmpeg PATH 등록

```powershell
# FFmpeg 설치 여부 확인
ffmpeg -version       # → 명령어 없음
where.exe ffmpeg      # → 찾을 수 없음

# winget 설치 시도
winget install Gyan.FFmpeg
# → "이미 설치된 패키지" (버전 8.1.1), PATH만 미등록 상태

# 설치 경로 탐색
Get-ChildItem "$env:LOCALAPPDATA\Microsoft\WinGet\Packages" -Filter "*ffmpeg*"
# → C:\Users\codelab\AppData\Local\Microsoft\WinGet\Packages\Gyan.FFmpeg_...\ffmpeg-8.1.1-full_build\bin

# 현재 세션 PATH 추가
$env:PATH = "C:\Users\codelab\AppData\Local\Microsoft\WinGet\Packages\Gyan.FFmpeg_Microsoft.Winget.Source_8wekyb3d8bbwe\ffmpeg-8.1.1-full_build\bin;$env:PATH"

# 사용자 환경변수 영구 등록
[Environment]::SetEnvironmentVariable("PATH", "$ffmpegBin;$userPath", "User")

# 동작 확인
ffmpeg -version
# → ffmpeg version 8.1.1-full_build-www.gyan.dev

# FFmpeg PATH 포함하여 백엔드 재시작
python -m uvicorn app.main:app --host 127.0.0.1 --port 8000
curl http://127.0.0.1:8000/health
# → {"status":"ok","env":"development"}
```
