## 원인

`backend/app/services/media.py`의 `audio_metadata()` 함수에서 `if audio:` 조건 사용.

mutagen의 `File()` 객체는 ID3 태그가 없는 MP3 파일을 읽을 때 **빈 딕셔너리처럼 동작**하므로 `bool(audio) == False`가 됩니다. 이 때문에 `audio.info.length`(실제 재생 시간)가 존재함에도 duration이 0.0으로 저장됨.

```python
# 버그 코드
if audio:           # 태그 없으면 False → info를 읽지 않음
    if audio.info:
        duration = ...

# 수정 코드
if audio is not None:  # None 여부만 체크
    if audio.info:
        duration = ...
```
