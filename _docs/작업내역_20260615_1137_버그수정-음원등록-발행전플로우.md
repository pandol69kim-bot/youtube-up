## 변경된 파일

- `backend/app/services/media.py`: `audio_duration` 제거, `audio_metadata` 추가 (duration + title/artist/album ID3 파싱 통합)
- `backend/app/routers/tracks.py`: `audio_duration` → `audio_metadata` 교체, 업로드 시 아티스트/앨범 자동 추출
- `backend/app/routers/video.py`: 미사용 임포트 3개 제거 (`Thumbnail`, `ThumbnailRequest`, `generate_thumbnail`)
- `frontend/app/page.tsx`: 버그 2건 수정
  - `refreshPlaylists`: 새로고침 버튼 클릭 시 기존 selectedPlaylist의 트랙 목록도 갱신하도록 수정
  - `updateTrack` → `handleTrackChange` + `saveTrack` 분리: onChange는 로컬 상태만 업데이트, onBlur에서 API 호출 (키 입력마다 API 호출하던 문제 및 응답 순서 역전 버그 해결)

## 주요 결정 사항

- ID3 파싱에 `easy=True` 파라미터 사용: MP3/FLAC/M4A 등 포맷별 키 이름 차이를 EasyTag 인터페이스로 통일 (`"title"`, `"artist"`, `"album"`), 태그가 없는 파일은 파일명 기반 fallback 유지
- 트랙 편집 저장 방식: 디바운스 대신 onBlur 방식 채택 — 타이핑 완료 후 포커스 이탈 시 저장, 구현 단순성 우선
